# homebrew-diskspace

Homebrew tap for [diskspace](https://github.com/tymrtn/diskspace) — *find the dead weight in your cargo hold.*

## Install

```bash
brew tap tymrtn/diskspace
brew install diskspace
```

Or one-liner:

```bash
brew install tymrtn/diskspace/diskspace
```

## Upgrade

```bash
brew upgrade diskspace
```
